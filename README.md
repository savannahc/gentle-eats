# Kenko

A non-addictive meal tracker focused on how food makes you feel, not calorie counting.

- 📷 Snap photos of meals (stored in-app, never your camera roll)
- 🍽 Rate hunger before and satisfaction after (1-3 scale)
- 🎙 Voice notes via Web Speech API
- 📊 Soft metrics: hunger patterns, satisfaction trends, time-of-day habits
- 📱 Works offline, installable as PWA

Built with vanilla HTML/CSS/JS. No frameworks, no backend, no tracking.
